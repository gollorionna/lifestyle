# lifestyle 
A website that brings together a community of people who love a healthy lifestyle.
This project was created as part of a web development internship. It includes practice with HTML/CSS.

## How to run a project locally
1. Clone repository:
   git clone [https://gollorionna.github.io/lifestyle/](https://github.com/gollorionna/lifestyle.git)
2. Navigate into the project folder:
   cd livestyle
3. Open index.html directly in your browser.

## Technologies Used 
*HTML5
*CSS3 (Flexbox)
